# example-marketplace
Projeto realizado com intuido de prototipar um software com funcionalidades basicas de um markteplace.
O mesmo foi usado Html5, CSS3, Javascript, jQuery e Bootstrap.

**Paginas construidas:**
- Carrinho
- Cadastro de produto
- Edição de produto
- Detalhes de produto
- Cadastro de vendedor
- Edição de vendedor
- Checkout
- Login
- Pesquisa
